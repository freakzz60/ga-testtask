Автоматическая выгрузка котировок USD/RUB и JPY/RUB в Excel, форматирование и расчеты в соответствии с тестовым
заданием, отправка выгруженного файла на электронную почту.

Краткое описание:
1. Запуск по требованию;
2. С сайта moex.com получаются котировки по указанным валютным парам за прошлый месяц;
3. Формируется и форматируется Excel-таблица, производятся необходимые вычисления;
4. Сформированный файл отправляется на указанную электронную почту.

Для работы, в конфигурации email.ini необходимо указать адрес почтового сервера для отправки письма, обратный адрес,
логин и пароль аккаунта-отправителя, адрес получателя.

Запуск по launch.bat. Среда выполнения - Python 3.12 с библиотеками из requirements.txt
